
# Question 32
=============
### Difference between Abstract class and Interface - given a situation what would you choose between abstract class and interface?
--------------------------------------------------------------------
* Des méthodes peuvent être implémentées dans les classes abstraites alors que l'interface est un contrat définissant des méthodes.
* Une classe ne peut hériter que d'une autre classe mais peut implémenter plusieurs interfaces
* Les classes abstraites sont utilisées dans les patterns comme "template" ou "stratégie" afin de définir des comportements ou des algorithmes pour les classes filles.
* L'utilisation d'interface permet un couplage lâche, une extension plus facile des programmes.
