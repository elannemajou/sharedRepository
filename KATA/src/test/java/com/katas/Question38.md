
# Question 38
=============
### What is coupling? How do you reduce coupling and why?
--------------------------------------------------------------------
* Le couplage est est la façon dont la modification d'une classe ou d'un fonctionnement dans le programme impacte une autre classes ou un groupe de classe.
* C'est aussi la quantité de code qu'il faudra modifier pour que le programme s'enrichisse d'une nouvelle fonctionnalité ou pour que son comportement change.
